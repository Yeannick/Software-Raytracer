# Software-Raytracer
software raytracer with PBR materials


![alt text](https://github.com/Yeannick/Software-Raytracer/blob/main/Raytracer.png)


- ray hit detection on :
  - Planes
  - Spheres
  - Triangles

- Possible Materials :
  - Lambert
  - Lambert phong
  - PBR 
all materials use BRDF.

- Perspective Camera
- Point Light
- Directional light
